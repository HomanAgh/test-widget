Widget Implementation Guide

This page demonstrates the direct script implementation of EliteProspects widgets. This method embeds widgets directly on your page (without iframes) for better integration and performance.

Basic Implementation

<!-- alumni Widget -->
        <div 
          class="ep-widget" 
          data-widget-type="alumni" 
          data-selected-teams='[{"id":"18741","name":"San Jose Barracuda"}]'
          data-selected-leagues='["nhl","shl","ahl","khl","nl","liiga","czechia","del","echl","icehl","slovakia","hockeyallsvenskan","ncaa","usports","acac","acha","ohl","whl","ushl","qmjhl","j20-nationell","mhl","cchl","pwhl-w","sdhl-w","nwhl-ca-w","phf-w","ncaa-w","ncaa-iii-w","acha-w","acha-d2-w","jwhl-w"]'
          data-include-youth="false"
          data-selected-league-categories='{"junior":true,"college":true,"professional":true}'
          data-background-color="#052D41"
          data-text-color="#000000"
          data-table-background-color="#FFFFFF"
          data-header-text-color="#FFFFFF"
          data-name-text-color="#0D73A6"
        ></div>

        <!-- Widget Loader Script (place at the end of the body) -->
        <script src="https://widget.eliteprospects.com/widget-loader-combined.js"></script>
